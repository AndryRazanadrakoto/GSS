# Installation
- Installer PHP / MySQL
- Créer un répetoire GSS dans www ou htdocs, selon le serveur web
- Copier dans ce répertoire toute l'arborescence de GSS
- Créer la base de donnée avec "GSS\sql\Creation_base_GSS.sql" (-> crée une base de données nommée "gss")
- Sur cette base de donnée, créer un utilisateur "gss", mot de passe "sql_gss" ayant les droits SELECT, INSERT, UPDATE, DELETE sur la base nommée "gss" :
	GRANT USAGE ON *.* TO 'gss'@'localhost' IDENTIFIED BY PASSWORD '*0D9E6D3D7BAA04D71C07EAE79ABC32270622E950';
	GRANT SELECT, INSERT, UPDATE, DELETE ON `gss`.* TO 'gss'@'localhost';
	Modifier éventuellement les 2 "localhost" de ces 2 lignes pour les remplacer par le serveur qui accèdera à la base (dans le cas ou le serveur PHP est différent du serveur MySQL)
- Modifier le fichier "configuration.ini" :
	- Utilisation du LDAP ou non
	- Host de la BDD
- Lancer avec http://host:port/GSS/
